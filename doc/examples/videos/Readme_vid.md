# Examples on how to use CurveSimulator

Find all example files in the CurveSimulator repository under doc/examples/.

| Subdirectory | Filename         | Description                                                                                                                                                                                                                                                                                                                     |
|--------------|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| TOI4504      | TOI4504_V001a... | Entire TESS observation period from 2018 to 2026. The full video is not particularly interesting unless you want to show the star wobbling, but a single frame can be used to display the light curve and the RV curve.                                                                                                         |
| TOI4504      | TOI4504_V001b... | Identical to video 1a, but contains only the light curve and the RV curve.                                                                                                                                                                                                                                                      |
| TOI4504      | TOI4504_V002...  | One complete orbit of Planet c.                                                                                                                                                                                                                                                                                                 |
|              |                  | Videos 3, 4, and 5 show the development of the transits of Planet d. All bodies are shown to scale so that it is accurately represented whether the bodies eclipse each other. In the other videos, the planets are shown at a larger scale to make them easier to recognize. The scale can be adjusted flexibly in each video. |
| TOI4504      | TOI4504_V003...  | No transit at the start of the TESS observations (2018).                                                                                                                                                                                                                                                                        |
| TOI4504      | TOI4504_V004...  | First grazing transit (2023).                                                                                                                                                                                                                                                                                                   |
| TOI4504      | TOI4504_V005...  | Full transit (2026).                                                                                                                                                                                                                                                                                                            |
| TOI4504      | TOI4504_V006...  | A single video frame corresponds almost exactly to the mean osculating period of Planet c, creating a stroboscopic effect. This allows us to learn a great deal about the relative positions and interactions of Planets c and d. The video depicts a real-time span of 23 years.                                               |
| TOI4504      | TOI4504_V007...  | Identical to video 2 but also includes Planet b. This makes it possible to see the influence of Planet b on the RV curve as well.                                                                                                                                                                                               |
