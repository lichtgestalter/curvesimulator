# Examples on how to use CurveSimulator

Find all example files in the CurveSimulator repository under doc/examples/.

| Subdirectory | Description                                                                                                                                   |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| 1            | Single run: Runs a single simulation, generates a video, and compares the computed flux, radial velocity and transit times with observations. |
| 2            | LMFit: Least-squares fit to find the parameter set that best fits the observed transit times.                                                 |
| 3            | LMFit: Least-squares fit to find the parameter set that best fits the observed flux.                                                          |
| 4            | MCMC: Markov Chain Monte Carlo fit for robust Bayesian parameter estimation, fitting the observed transit times.                              |
| 5            | MCMC: Markov Chain Monte Carlo fit for robust Bayesian parameter estimation, fitting the observed flux.                                       |
